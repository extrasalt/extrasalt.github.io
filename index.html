Circle
A file organizer for everyone

From reading 'The Life-Changing Magic of Tidying', I learnt how reducing clutter can significantly improve our lives. Since I spend most of my time on a computer, I decided to write an application that would organize cluttered folders like downloads and desktop.

I present to you: Circle, A file management application that can intelligently find related files and group them together. This version supports grouping based on file type. Circle also has a 'Custom' mode that let's you group files that match a given pattern.

Circle is open-source and is one of the Gopher Gala 2015 finalists.
